## Setup

```bash
git clone git@github.com:gdborton/programming-game-starter.git
cd programming-game-starter
touch .env
# Update your .env with USER_ID and API_KEY
# You can find these values here:
# https://programming-game.com/dashboard
npm install
npm start
```
